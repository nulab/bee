# bee

A CLI for [Backlog](https://backlog.com/) project management service, inspired by GitHub CLI.

> [!NOTE]
> This is not an officially supported Nulab product. It is maintained by volunteers.

## Install

```sh
npm install -g @nulab/bee
```

## Quick Start

Authenticate with your Backlog space:

```sh
bee auth login
```

Then start using bee:

```sh
bee issue list
bee issue view PROJECT-123
bee project list
```
