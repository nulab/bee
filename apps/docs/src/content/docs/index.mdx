  </Card>
  <Card title="安全なマルチスペース対応" icon="setting">

    プロジェクトごとに最適な認証で運用できます。
  </Card>
  <Card title="AI エージェントと連携" icon="puzzle">

    Claude Code などの AI エージェントが bee の使い方を自動的に習得。

    簡単な指示だけで、Backlog の課題作成やステータス更新をこなせます。
  </Card>
  <Card title="CI/CD に組み込む" icon="github">
    [GitHub Actions や GitLab CI から](/bee/integrations/ci-cd/) Backlog の課題ステータスを自動更新。

    デプロイと課題管理をシームレスにつなぎます。
